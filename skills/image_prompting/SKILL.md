# Image prompting

- Specify subject, setting, lighting, camera angle, and color palette; avoid celebrity likenesses or trademarks.
- Align the metaphor with the slide title; keep text in-image minimal (prefer none).
- If the treatment is `uploaded_image` or `pdf_figure`, refine alt text for accessibility instead of inventing a new scene.
- For `generate_mock` or diffusion, prefer flat vector / editorial illustration styles for B2B LinkedIn.
